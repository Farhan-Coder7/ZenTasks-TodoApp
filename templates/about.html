About ZenTasks

ZenTasks is a minimal and efficient to-do list web application designed to help you manage your daily tasks with clarity and focus. Built using Flask for the backend and Bootstrap 5 for the frontend, it combines simplicity with functionality.

Key Features
  • Create, update, and delete your tasks easily
  • Responsive and clean user interface
  • Lightweight and fast performance
  • Secure database integration using SQLAlchemy
  • Easy to use and accessible from any device
